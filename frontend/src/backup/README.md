# HOMU 백업 파일

**백업 일시**: 2026년 1월 9일

## 백업된 파일 목록

### 1. App.tsx
- 메인 애플리케이션 컴포넌트
- 라우팅 및 다크모드 관리
- 헤더 스크롤 애니메이션
- 아파트 상세 페이지 네비게이션

### 2. Favorites.tsx (최신 버전)
- 즐겨찾는 지역 탭: 파주시, 강남구, 화천군 지역별 시세 분석
- 즐겨찾는 매물 탭: 카테고리별 아파트 목록
  - 💎 시세 높음: 강남 타워팰리스 등 고가 아파트 5개
  - 💰 시세 저렴: 파주 문산, 화천 등 저가 아파트 5개
  - 🛡️ 안전 매물: 전세가율 60% 이하 안전 아파트 5개
  - 🔥 거래 활발: 최근 거래량이 많은 아파트 5개

## 주요 기능

### 즐겨찾기 페이지 카테고리 시스템
- 2x2 그리드 레이아웃의 카테고리 버튼
- 각 카테고리별 5개 아파트 목록
- 전세가율 기반 4단계 안전도 표시 (안전/보통/주의/위험)
- 거래 활발 카테고리는 거래 건수 표시
- 부드러운 애니메이션 전환 효과

### 전세가율 안전도 기준
- 안전: 60% 미만 (녹색)
- 보통: 60-70% (파란색)
- 주의: 70-80% (노란색)
- 위험: 80% 이상 (빨간색)

## 기술 스택

- Recharts (차트)
- Tailwind CSS v4
- Lucide React (아이콘)
